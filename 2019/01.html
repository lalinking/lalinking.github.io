前提

数据库表名、字段名均采用蛇形命名（即以下划线分隔单词）。 Java 类名、字段名均采用驼峰命名。 要求 Java - 数据库之间能互相转换。如果不能转换也不用急，可以调整部分代码，建议在PO类上添加注解进行映射）

原理

配置一个 mybatis Interceptor，利用 Mapper 接口中方法的声明来动态生成 SQL。 任何加了这些注解的方法都会进入拦截器：@Insert("C")、@Update("U")、@Select("R")、@Delete("D")
